## Getting Started

To get started with the project, follow these steps:

1. Install the project dependencies by running the following command in your terminal:

```bash
yarn install
```

2. Once the dependencies are installed, start the development server by running:

```bash
yarn run dev
```

3. After the server starts, open your browser and navigate to [http://localhost:3000](http://localhost:3000) to see the result.

Feel free to explore the project and make any necessary changes to suit your needs.
